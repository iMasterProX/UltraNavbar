# UltraNavbar
![Guide](guide.png)

## Overview
UltraNavbar is a custom navigation bar built for tablet devices. It runs as an accessibility overlay and focuses on reliable navigation controls, home screen backgrounds, and configurable behaviors for large screens.

## Features
- Custom navigation bar overlay with back, home, recents, and quick actions
- Home screen background preview and apply with optional filter
- Per-app hide modes (blacklist/whitelist) and fullscreen auto-hide
- Hotspot area to reveal the bar when hidden
- Accessibility service integration with boot and settings handling
